GlobalStyle
===========

### Import
```js
  import GlobalStyle from '@govuk-react/global-style';
```
<!-- STORY -->


A Styled Component to apply global style for use with govuk-react.

### Usage

Simple

```jsx
import GlobalStyle from '@govuk-react/global-style';

const MyApp = () => <>
  <GlobalStyle />
  <div>Example</div>
</>
```
