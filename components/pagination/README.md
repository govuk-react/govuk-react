Pagination
==========

### Import
```js
  import Pagination from '@govuk-react/pagination';
```
<!-- STORY -->

### Usage

```jsx
import { Pagination } from 'govuk-react';

<Pagination>
  <Pagination.Anchor href="#prev" previousPage>
    Previous page
  </Pagination.Anchor>
  <Pagination.Anchor href="#next" nextPage>
    Next page
  </Pagination.Anchor>
</Pagination>
```
