Page
====

### Import
```js
  import Page from '@govuk-react/page';
```
<!-- STORY -->


### Usage

Simple

```jsx
import { Page, BackLink, H1 } from 'govuk-react';

<Page beforeChildren={<BackLink href="#" />}>
  <H1>Page title</H1>
</Page>
```
